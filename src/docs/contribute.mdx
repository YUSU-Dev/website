# Contributing
Welcome and thank you for contributing to York SU website. This guide will take you through the technical considerations for contributing.

## Code of Conduct
The York SU has adopted the [Contributor Covenant](https://www.contributor-covenant.org/).

## How to contribute

### Raise a bug or suggest an improvement

1. Raise a bug or suggest an improvement by creating an issue in the [GitHub repository](https://github.com/YUSU-Dev/Vue-3-Components/issues).
2. Internal users can raise a bug or suggest an improvement by emailing the IT team.
3. The IT team will review the issue and get back to you.

### Contribute to the codebase

You can help speed up the development of the YorkSU website by contributing to the codebase. Here's how you can do it:
1. Select an issue from the [GitHub repository](https://github.com/YUSU-Dev/Vue-3-Components/issues).
2. If an issue doesn't exist, create one.
3. Fork the repository and create a new branch.
4. Make your changes and commit them.
5. Push your changes to your fork.
6. Create a pull request to `main` for review.
